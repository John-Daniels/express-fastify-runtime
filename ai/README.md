# ai/

This folder holds context for working with AI on this project from anywhere.

- **CONTEXT.md** — Single file with all "memories": architecture, rules, structure, what's supported, optimization principles, listen/ServerLike, commands, and how to continue. Give this file (or its path) to any AI assistant when you start a session so it has full project context.
- **context.json** — Same context in JSON for tools, scripts, or any consumer that prefers structured data. Do not delete CONTEXT.md; both are kept in sync.
